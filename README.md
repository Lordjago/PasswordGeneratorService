# PasswordGeneratorService
PasswordGeneratorService is a simple express API for password generation 
## Installation
Use the package manager [npm](https://www.npmjs.com/package/) to install.

```bash
npm install
```
## Usage

```python
route : -

http://localhost:3000/generate-password

```
